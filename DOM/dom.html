DOM (Document Object Model) (Belge Nesne Modeli)

DOM, belge içeriğini belirli bir düzen içerisinde nesne olarak temsil edilmesini sağlayan bir modeldir

HTML DOM

HTML DOM, HTML için standart bir nesne modeli ve programlama arayüzüdür.

  • HTML ögelerini nesne olarak tanımlar
  • Tüm HTML ögelerinin özelliklerini(attribute) tanımlar
  • Tüm HTML ögelerinin yötemlerine(metot/event) erişir

HTML ögelerinin nasıl alınacağına, değiştirileceğine, ekleneceğine ve silineceğine ilişkin bir standarttır.

Nesne modeliyle Javascript, dinamik HTML oluşturmak için ihtiyaç duyduğu gücü alır.

  1. Javascript, sayfadaki tüm HTML ögelerini değiştirebilir
  2. Javascript, sayfadaki tüm HTML özelliklerini(attribute) değiştirebilir
  3. Javascript, sayfadaki tüm CSS sitillerini değiştirebilir
  4. Javascript, mevcut HTML ögelerini be niteliklerini(attribute) kaldırabilir
  5. Javascript, yeni HTML elementleri ve nitelikleri ekleyebilir
  6. Javascript, sayfadaki tüm mevcut HTML eventlerine tepki verebilir
  7. Javascript, sayfadaki elementlere yeni event oluşturabilir

Bir web sayfası yüklendiğinde, tarayıcı sayfanın bir Belge Nese Modeli (DOM) oluşturur

HTML DOM modeli, bir Nesneler ağacı olarak oluşturulur.
