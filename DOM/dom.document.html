DOM Document

HTML DOM document nesnesi tüm sayfamızın sahibi bir nesnedir

Method Description
document.getElementById(id) Elementi ID'si üzerinden bulur. Aynı ID'ye sahip başka elementler varsa sadece ilkini getirir
document.getElementsByTagName(name) Elementin tag adına göre elementleri bulur
document.getElementsByClassName(name) Elementin class adına göre elementleri bulur
Method Description
document.createElement(element) Element oluşturur
document.remove(element) Elementi siler
document.appendChild(element) Elementin altına element ekler
Method Description
document.getElementById(id).onclick = function() {code} Elemente event atar
Method Description
document.achors a elementlerden name attribute'üne sahip olanları listeler
document.baseURI Belgenin mutlak temel URI'sine ulaşmak istersem bunu kullanırım
document.body Body elementini dönderir
document.cookie Belgenin çerezini dönerir
document.forms Belgedeki tüm form elementlerini dönderir
document.scripts Tüm script elementlerini dönerir
document.URL Belgenin tam URL'sini dönderir
